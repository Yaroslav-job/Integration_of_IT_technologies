How to Play Sudoku

Sudoku is a logic-based number placement puzzle.

The objective is to fill a 9×9 grid with digits 1-9 so that each column, each row, and each of the nine 3×3 subgrids contain all the digits from 1 to 9.

Controls:

  • Click on a cell to select it
  • Press 1-9 on keyboard to enter a number
  • Press 0, Delete, or Backspace to clear a cell
